# nylira-maze

A WIP library for generating maze data.

# Install

    npm i nylira-maze --save

# Use

Basic usage:

    var maze = require('nylira-maze')

    // make a new 10x10 maze
    maze()

    // make a new 20x20 maze
    maze(20)

    // make a new 15x23 maze
    maze(15, 23)

Want to preview the grid?

    maze(15, 23, true)
